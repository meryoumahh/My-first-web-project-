Introduction to CSS

CSS is used to style HTML documents. It allows you to control the layout, fonts, colors, and other visual aspects of your website. CSS is a separate file from your HTML file, but it is linked to your HTML file using a<link>tag in the <head> section of your HTML file.

Selector Types

There are several types of selectors in CSS:

  • Element Selector - selects all elements of a certain type (e.g. <p>, <h1>, etc.)
  • Class Selector - selects all elements with a certain class (e.g. .my-class)
  • ID Selector - selects an element with a certain ID (e.g. #my-id)
  • Attribute Selector - selects all elements with a certain attribute (e.g. [href])
  • Pseudo-Class Selector - selects elements based on a certain state (e.g. :hover)

Cascading and Specificity

CSS stands for Cascading Style Sheets, which means that styles can cascade from one rule to another. This can be helpful in certain situations, but it can also cause problems if you don't understand how it works. Specificity is another important concept in CSS. It refers to the "weight" of a selector. For example, an ID selector is more specific than a class selector. If two selectors have the same specificity, the last one to appear in the CSS file will take precedence.

CSS Layout

Flexbox

Flexbox is a newer layout technique that allows you to create flexible layouts that adapt to different screen sizes.

Grid

CSS Grid is another newer layout technique that allows you to create complex, multi-dimensional layouts.

Positioning

CSS also has a few different positioning techniques, including absolute positioning, relative positioning, and fixed positioning. These can be useful for creating specific layouts or for positioning elements within a layout.

CSS Box Model

The CSS Box Model is a way of understanding how elements are laid out on a web page. It consists of four parts:

  • Content - the actual content of the element (e.g. text or an image)
  • Padding - the space between the content and the border
  • Border - a line around the element
  • Margin - the space between the border and the next element
